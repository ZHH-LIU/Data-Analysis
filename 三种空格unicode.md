## Unicode
* 不间断空格<br>
```python
\u00A0
```
主要用在office中,让一个单词在结尾处不会换行显示,快捷键ctrl+shift+space。
* 半角空格(英文符号)<br>
```python
\u0020
```
代码和英文文章中常用的。
* 全角空格(中文符号)<br>
```python
\u3000
```
中文文章中使用。
